# SMART-ATTENDENCE-MANAGEMENT
Smart Attendance Management System is a Java-based application designed to manage and track attendance efficiently. It uses Swing (JFrame) for the graphical user interface and SQLite for database management, providing a simple and user-friendly experience.
🛠️ Technologies Used
Java (Core Java)
Swing (JFrame) – for UI
SQL (SQLite) – for database
🚀 Features:
User Authentication
Faculty and Students can register and login securely into the system.
Code-Based Attendance System
Faculty generates a unique attendance code based on subject and date.
Real-Time Code Sharing
Generated code is displayed on the student portal for attendance submission.
Student Attendance Submission
Students enter the code and mark their attendance easily.
Attendance Tracking for Students
Students can view how many subjects’ attendance they have completed.
Faculty Dashboard
Faculty can monitor attendance records efficiently.
Subject-wise Attendance View
Faculty can check how many students are present and absent for each subject.
Date-wise Record Management
Attendance is stored and managed according to subject and date.
